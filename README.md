# To start your app please:

```
git clone  http://example.com/your-repo.git
cd your-repo
yarn install
yarn start

```

## To run tests:

```
yarn test
```

## To build your app:

```
yarn build
```

## To deploy your app:

```
yarn deploy
```

## To lint your app:

```
yarn lint
```
